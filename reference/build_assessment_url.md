# Build assessment URL

Constructs the URL for downloading assessment data files based on year
and pattern.

## Usage

``` r
build_assessment_url(end_year, subject, grade, level)
```

## Arguments

- end_year:

  School year end

- subject:

  "ela" or "math"

- grade:

  Grade level

- level:

  "district" or "school"

## Value

URL string
