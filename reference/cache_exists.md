# Check if cached data exists and is valid

Check if cached data exists and is valid

## Usage

``` r
cache_exists(key, max_age = 30)
```

## Arguments

- key:

  Cache key (e.g., "enr_tidy_2024" or "graduation_2024")

- max_age:

  Maximum age in days (default 30)

## Value

TRUE if valid cache exists
