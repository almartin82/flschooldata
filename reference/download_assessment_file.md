# Download a single assessment file

Downloads and parses a single assessment Excel file from FLDOE.

## Usage

``` r
download_assessment_file(end_year, subject, grade, level)
```

## Arguments

- end_year:

  School year end

- subject:

  "ela" or "math"

- grade:

  Grade level

- level:

  "district" or "school"

## Value

Data frame with assessment data
